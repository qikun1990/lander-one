# complex-vue-form
Complex Vue form with:
* Router
* Vuex
* Validation
* Tabs
* Accordion
* Rich-text editor
* Multiselect

Project page: [Complex Vue form](https://yegupov.github.io/complex-vue-form/).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
